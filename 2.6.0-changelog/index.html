RetroPage 2.6.0 Changelog

Retroity · September 27, 2020

This is the changelog for the first update to RetroPage in 2 years, RetroPage 2.6.0. You can download the release here.

Changes and Improvements

  • ixquick.eu and Searx have been removed as Search Engine options.
  • Ecosia has been added as a search engine option.
  • Keyboard Shortcut functionality has been removed due to a general lack of usefulness, and due to said shortcuts often conflicting with operating system or web browser shortcuts.
  • The RetroPage blog has been removed from RetroPage itself and has become a separate repository.
  • The "Tech" section has been renamed to "Tech News"
  • Several links have been replaced.
  • The copyright year listed have been updated to 2020.
  • "Source" files have been removed and Javascript and CSS files (with the exception of CSS/Javascript files for libraries) are no longer minified.
  • The contributor guidelines have been updated.
