RetroPage 2.3.0 Changelog

Retroity · February 5, 2018

This is a small update for Retropage that changes and adjusts a few things. You can find this update's GitHub release here.

Changes and Improvements

  • Ixquick.eu is now included as a search engine
  • The footer now displays the current RetroPage version
  • Replaced the "Classic Noscript Version" footer link with "Javascript Version" in the noscript version
  • Due to it no longer being in operation, links to Vidme have been replaced with Amazon Prime Video.
  • Links to Gab.ai have been replaced with Minds.
  • Updated Newgrounds links to use HTTPS, due to it supporting HTTPS now.
